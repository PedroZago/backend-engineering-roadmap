# NestJS Clean Architecture API

Enterprise-grade backend API built with NestJS, following Clean Architecture principles.

## Goals

- Build production-ready architecture
- Implement authentication and authorization
- Apply caching strategies
- Implement testing pyramid
- Deploy using AWS

## Key Features

- JWT authentication
- Role-based access control (RBAC)
- PostgreSQL with migrations
- Redis caching
- Centralized logging
- Unit and integration tests
- CI/CD pipeline
- AWS deployment

## Architecture

- Domain Layer
- Application Layer
- Infrastructure Layer
- Interface Layer
